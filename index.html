Each threshold changes one variable at a time while holding all other settings constant. “No switch” means GOMINING Lock did not become the top reward choice anywhere within the tested range.
The dashboard uses your actual GoMining daily pool reward, electricity, service, net reward, and current discount. It first checks that pool reward minus electricity and service reproduces the app’s net reward. It then compares BTC, TH, GOMINING wallet, GOMINING lock, and efficiency upgrades in BTC-equivalent terms. Assumptions remain editable because future BTC price, token price, hashprice, and resale values cannot be known.
